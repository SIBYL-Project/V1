SIBYL - (Analisis Data + Sinyal Simulasi)

⚠️ PERINGATAN RISIKO PENTING 
Sebelum menggunakan SIBYL, Anda harus memahami dan menyetujui bahwa:
  • Akurasi analisis sangat bergantung pada kualitas data yang dimasukkan
  • Sistem tidak membaca berita ekonomi, kinerja perusahaan, atau manipulasi pasar
  • Statistik tidak dapat mendeteksi peristiwa tiba-tiba (Black Swan Event)
  • Hasil analisis hanya untuk tujuan edukasi dan alat bantu keputusan
